# Contributing to Awesome AI Agents Security

First off, thank you for considering contributing to Awesome AI Agents Security! It's people like you who make this tool an excellent resource for the community.

To ensure this list remains high-quality and easy to navigate, please adhere to the following guidelines:

### 1. Scope
* **Relevance:** The tool or resource must be directly related to the **security of autonomous AI agents** (e.g., runtime protection, sandboxing, red teaming agents).
* **Open Source:** We prioritize open-source tools. If you are adding a proprietary tool, please ensure it offers a significant free tier or is a widely recognized industry standard.
* **Maintenance:** Please do not submit valid "abandonware." Tools should have had a commit within the last 12 months.

### 2. Search First
Before you submit a pull request, please check the existing list to ensure the tool hasn't already been added.

### 3. Formatting
Please follow this format for adding new entries:

```markdown
- **[Tool Name](URL)** - A concise description of what the tool does. Ends with a period.
```
